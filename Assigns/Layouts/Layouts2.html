Introductory Android

(Layouts and Activities)

Overview

This and later exercises will draw on the introductory courses supplied at

https://developer.android.com/courses

and for the first homeworks, this course in particular:

https://developer.android.com/courses/android-basics-kotlin/course

Each of these courses is somewhat "code by the numbers" overview of Android dev topics. To use them effectively, note:

  • The videos are mostly just cute rah-rah, so you can skip those.

  • The codelab walkthroughs are where the serious content is. For these, actually write the code, don't just copy-paste it. As you write each line, be sure you understand what it does and why.

  • Where feasible, first *guess* what the code should be and then read their suggested code.

  • Explore "around" the code, trying different alternatives to see how they work. I'll help with this by assigning required homework explorations surrounding the code in the walkthroughs.

  • Do the quiz at the end of each path, and please report as part of your homework submission how well you did on it.

  • A canned example like these codelabs, however well design point of this question is to do, at least for one case, independent e(E.g. deeper exploration of one or more of the classes used, of Gradle, of other widgets, etc.).
